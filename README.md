# 🏛️ The MET Museum Landing Page

An adaptive, single-page website (Landing Page) for the legendary **The Metropolitan Museum of Art**. This project was built to practice modern web development workflows, pixel-perfect semantic layout, and BEM methodology.

## 🛠️ Tech Stack

*   **HTML5** — Semantic markup for accessibility and SEO.
*   **SCSS / CSS3** — Flexbox and CSS Grid for robust layouts.
*   **BEM Methodology** — Clean and maintainable class naming architecture.
*   **JavaScript (ES6+)** — Interactive mobile navigation menu and functional components.

---

## 📱 Features

*   **Fully Responsive:** Seamless layout optimized for mobile, tablet, and desktop viewports.
*   **Seamless Mobile Menu:** An interactive, full-screen mobile menu with smooth transition animations.
*   **Pixel Perfect:** Precise alignment of components and grids matching the original design.
*   **Micro-interactions:** Custom hover effects and focus states on all interactive elements (buttons, links, icons).
